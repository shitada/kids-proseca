# とうきょう でんしゃビート！

[![Deploy to GitHub Pages](https://github.com/shitada/kids-proseca/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/shitada/kids-proseca/actions/workflows/deploy-pages.yml)

🎮 **プレイはこちら → [https://shitada.github.io/kids-proseca/](https://shitada.github.io/kids-proseca/)**

5〜10歳向けのiPad横画面用リズムゲームです。画面上から走ってくる電車が駅へ着く瞬間に、光る線路をタップして音をつなぎます。

JR・私鉄等・地下鉄を巡る全13ステージを、1から順番に解放して遊べます。ステージが進むと線路は2本から最大6本まで増え、長押し・同時押し・高速パターンが少しずつ加わります。

各ステージのタップ音と伴奏は同じメジャースケール7音だけから生成され、路線ごとに異なる調と雰囲気を楽しめます。車両と音楽は権利確認前の開発用プレースホルダーです。

## 開発

```bash
npm install
npm run dev
```

## 確認

```bash
npm test
npm run build
```

キーボードではレーン数に応じて `S` / `D` / `F` / `J` / `K` / `L` を使用します。2レーン時は `F` / `J` です。

## 技術

- Three.js
- TypeScript
- Vite
- Web Audio API
- Vitest

## セーブ

ステージの解放状況、ベストスコア、獲得したきっぷはブラウザ内に保存されます。旧1ステージ版のクリア記録は新しい保存形式へ自動移行されます。

## 権利方針

実在するロゴ、忠実な車両外観、発車メロディ、録音音源は、権利者の許諾が確認できるまで公開用ビルドへ含めません。未許諾のメロディを音色だけ変えて再現することもせず、別旋律のオリジナル生成音へ差し替えます。
